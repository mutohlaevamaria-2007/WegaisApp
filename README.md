# ТЗ
Необходимо написать приложение на языке  c# с используемым стеком:
1. WPF(xaml)
2. Работа с БД запросами Entity Framework
3. БД использовать SQL CE. Структуру БД необходимо построить с использованием подхода Code First.
4. В БД должна использоваться хотя бы 1 дополнительная миграция.
5. Написать несколько прямых SQL запросов.
6. Несколько запросов должно быть с помощью LINQ

Большим плюсом будет использоваение паттерна MVVM. 
Приложение должно выполнять следующее:
Парсинг файлов
1.Парсить файлы, которые приложены в папке.
2.Модели сущностей должны быть расписаны согласно структуре этих xml файлов.
3.Выводить на экран информацию о товаре, производителе и количестве списком или таблицей.
4.Сохранять в БД. Структура БД должна учитывать связи по первичным ключам.
Запрос информации о погоде
Запросить информацию о погоде любого города(например Уфа) используя WEB запросы (REST API).
Вывести на экран минимальную информацию о погоде.
